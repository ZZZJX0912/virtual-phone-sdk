开发中…
一个提供录音、播放方法的小库


## 结构初设：
  - audioSource：**音频输入**
    - 振荡器
    - 流（录音流，文件流）
    - audio标签
    - 内存音频（audioBuffer）
  - audioEffects：**音频处理**
    - 滤波器
    - 平移
    - 混响
    - 压缩
    - 切片发送？（提供用户传入回调，WebSocket？其他协议？）
    - 可视化声音？（音频波纹）
